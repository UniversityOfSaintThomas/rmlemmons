ADDITIONAL EXERCISES IN MIXED CONDITIONAL ARGUMENTS

1:  The book is too expensive; because if it costs more than ten dollars, it is too expensive; and, it does costs more than ten dollars.
 

2. Since Socrates considered doing philosophy a duty; so, he would not quit. For if Socrates had a duty, then he would not quit.
 

3. The soul is immortal. For if it is immaterial, then immortal; and, the soul is not immaterial.
 

4. If the soul does not exist, then there is no self. But there is no self. Thus the soul does not exist.
 

5. If conformity is more important than virtue, then Socrates should have abandoned his philosophical mission. But conformity is not more important than virtue. Thus Socrates should not have abandoned his philosophical mission.
 

6. Compassion involves being kind; because if compassion did not involve being kind, then Buddha was not compassionate. But Buddha was compassionate.
 
 

7.  Christmas involves seeking to be kind to others by giving them a gift; because, if Santa Claus was a woman, then Christmas would not involve seeking to be kind to others by giving them a gift, and Santa Claus was not a woman.
 
 

8. If X, then p. But p. Thus X
 

9. If not q, then not B. But B. Thus, q.
 

10. If not w, then not u. But not u. Thus not w.
 

11.  If not t, then not m. But m. Thus t.
 

12. If not F, then not G. Not F. Thus not G.
