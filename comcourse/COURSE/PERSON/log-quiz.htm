Logic Quiz: Identify the argument as modus ponens, modus tollens, disjunctive or biconditional. Indicate whether it is valid.

1: If x, then b. But b. Thus x.
 
 

2: If and only if Angels fly, then God exists. God exists. Thus angels fly.
 
 

3. Either the dog or the cat ate the cheese. The dog was not home. So the cat ate the cheese.
 
 

4. The farmer will plant, if the ground is not soggy. The ground is soggy. Thus, the farmer will not plant.
 
 

5. If it rains, then school is cancelled. It is raining.Thus school is cancelled.
